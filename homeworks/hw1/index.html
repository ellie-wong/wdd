Reading Responses

  1. There are so many design decisions that go into every app or website we view that we take for granted! Before this article, I never really considered how an app like Twitter might make decisions to change the length/detail of a timestamp based on which view one is looking at.
  2. My favorite part of this article were the GIFs deconstructing the design components of app pages. Clearly, the inclusion of these were a design choice in itself which allows the reader's eyes to be drawn to each individual element and how it fits into the app's design as a whole.
  3. "Who are the people using this product? How might their use cases be different from yours? How does the design serve these use cases?"
  4. I would rate this a 8/10 for its ease of read.
